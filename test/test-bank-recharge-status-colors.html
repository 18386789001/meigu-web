银行充值记录状态颜色修改测试

API接口信息

接口地址: https://jpmx.xyz/api/c2cOrder/list?page_no=1&direction=recharge&language=zh-CN

请求方法: GET

状态代码: 200 OK

功能: 银行充值记录列表

修改内容说明

在 wap-vue/src/views/Record/DepositAndWithdrawal.vue 页面中，为银行充值记录的状态文本添加了颜色显示：

  • 成功状态: 绿色 (#28a745)
  • 失败状态: 红色 (#EA0F0F)
  • 确认中状态: 黄色 (#F5D658)

修改的具体内容

1. 模板修改

<div class="status flex"> <template v-if="getStatusValue(item) == 0 && !isEthConverted(item)"> <div class="status-icon status-icon-color3 mr-2"></div> <span class="status-text status-text-pending">{{ $t('confirming') }}</span> </template> <template v-if="getStatusValue(item) == 1 || isEthConverted(item)"> <div class="status-icon status-icon-color1 mr-2"></div> <template v-if="isEthConverted(item)"> <span class="status-text status-text-success">{{ $t('ethConvertedToUsdtNotice') }}</span> </template> <template v-else> <span class="status-text status-text-success">{{ $t('success') }}</span> </template> </template> <template v-if="getStatusValue(item) == 2 && !isEthConverted(item)"> <div class="status-icon status-icon-color2 mr-2"></div> <span class="status-text status-text-failed">{{ $t('fail') }}</span> </template> </div>

2. CSS样式添加

// 状态文本颜色 .status-text { font-size: 14px; } .status-text-success { color: $green; } .status-text-failed { color: #EA0F0F; } .status-text-pending { color: #F5D658; }

状态显示效果

成功状态

成功
API返回: state = 1
显示效果: 绿色圆点 + 绿色"成功"文字

失败状态

失败
API返回: state = 2
显示效果: 红色圆点 + 红色"失败"文字

确认中状态

确定中
API返回: state = 0
显示效果: 黄色圆点 + 黄色"确定中"文字

颜色规范

颜色值定义

成功状态: 绿色 (#28a745) - 使用SCSS变量 $green
失败状态: 红色 (#EA0F0F) - 直接使用颜色值
确认中状态: 黄色 (#F5D658) - 直接使用颜色值

适用范围

修改范围

银行充值记录: ✅ 已添加状态文本颜色
银行提现记录: ✅ 已添加状态文本颜色
数字货币充值记录: ✅ 已添加状态文本颜色
数字货币提现记录: ✅ 已添加状态文本颜色
ETH转换状态: ✅ 使用成功状态的绿色

实现要点

  • ✅ 为所有状态文本添加了对应的颜色类
  • ✅ 保持了原有的状态图标颜色不变
  • ✅ 状态文本颜色与状态图标颜色保持一致
  • ✅ 支持多语言状态文本的颜色显示
  • ✅ 同时适用于银行和数字货币的充提记录
  • ✅ ETH转换状态也使用成功状态的绿色

状态映射

状态值映射

银行充值记录 (c2cOrder):
- state = 0 → 确定中 (黄色)
- state = 1 → 成功 (绿色)
- state = 2 → 失败 (红色)

数字货币充提记录 (rechargeBlockchain):
- status = 0 → 确定中 (黄色)
- status = 1 → 成功 (绿色)
- status = 2 → 失败 (红色)
- failure_msg = "当前币种为eth直接完成订单" → ETH已兑换为USDT (绿色)
