# 教育中心页面UI重设计说明

## 设计理念

重新设计教育中心页面，采用现代化简约的黑色高级感风格，营造专业、高端、科技感强的金融交易教育环境。

## 设计特点

### 1. 黑色高级感主题
- **深色背景**：使用黑色渐变背景 `#0a0a0a → #1a1a1a`
- **高对比度**：白色明亮文字确保清晰可见
- **专业氛围**：符合现代金融教育平台的设计趋势

### 2. 现代化简约设计
- **简洁布局**：去除冗余元素，专注核心功能
- **清晰层次**：合理的信息架构和视觉层次
- **现代元素**：使用渐变、阴影、毛玻璃等现代设计元素

### 3. 精选教育内容
- **4个教育分类**：外汇交易、股票投资、加密货币、商品期货
- **6门热门课程**：从基础到高级的完整学习路径
- **完整信息**：课程时长、学员数量、评分等级等关键数据
- **品牌色彩**：每个分类和课程都有独特的色彩标识

## 设计细节

### 1. 颜色方案
- **主背景**：黑色渐变 `linear-gradient(135deg, #0a0a0a 0%, #1a1a1a 100%)`
- **卡片背景**：深灰色 `rgba(30,30,30,0.95)`
- **主要文字**：纯白色 `#ffffff`
- **次要文字**：半透明白色 `rgba(255,255,255,0.7)`
- **按钮渐变**：蓝紫色渐变 `#667eea → #764ba2`
- **按钮渐变**：粉红色渐变 `#f093fb → #f5576c`

### 2. 教育分类色彩标识
- **外汇交易**：蓝紫色 `#667eea`
- **股票投资**：深紫色 `#764ba2`
- **加密货币**：粉紫色 `#f093fb`
- **商品期货**：粉红色 `#f5576c`

### 3. 课程色彩标识
- **外汇交易基础入门**：蓝紫色 `#667eea`
- **技术分析进阶课程**：深紫色 `#764ba2`

### 4. 交互效果
- **悬停效果**：卡片悬停时向上移动8-10px

### 3. 语言切换监听
```javascript
// 语言切换监听
let languageWatcher = null;

onMounted(() => {
  // 监听语言变化
  languageWatcher = watch(() => locale.value, () => {
    nextTick(() => {
      console.log('语言已切换:', locale.value);
    });
  });
});

onUnmounted(() => {
  if (languageWatcher) {
    languageWatcher();
  }
});
```
