# 货币选择器显示简化修改报告

## 🔍 用户需求

### 需求描述
用户希望在货币选择器中显示货币代号（如USD、JPY、CNY）而不是完整的货币名称（如US Dollar、Japanese Yen、Chinese Yuan）。

### 当前显示问题
从控制台日志可以看到：
```
获取货币名称: USD, 语言: English, 结果: US Dollar
```

但在ActionSheet中仍然显示中文名称：
- 美元
- 日元  
- 人民币
- 港币
- 卢比
- 马币

## 🔧 修改方案

### 1. **简化货币名称映射函数** ✅

#### 修改前（复杂的多语言映射）
```javascript
const getCurrencyName = (currency, lang) => {
  const currencyNames = {
    'USD': {
      'Chinese': '美元',
      'English': 'US Dollar',
      'Japanese': '米ドル'
    },
    'JPY': {
      'Chinese': '日元',
      'English': 'Japanese Yen',
      'Japanese': '日本円'
    },
    // ... 更多货币映射
  }
  
  return currencyNames[currency]?.[lang] || currency
}
```

#### 修改后（直接返回货币代号）
```javascript
const getCurrencyName = (currency, lang) => {
  // 直接返回货币代号，不进行多语言转换
  console.log(`获取货币名称: ${currency}, 语言: ${lang}, 结果:`, currency)
  return currency
}
```

### 2. **显示效果对比**

#### 修改前的显示
```
$ US Dollar      (英文环境)
¥ Japanese Yen   (英文环境)
¥ Chinese Yuan   (英文环境)

$ 美元           (中文环境)
¥ 日元           (中文环境)
¥ 人民币         (中文环境)
```

#### 修改后的显示
```
$ USD            (所有语言环境)
¥ JPY            (所有语言环境)
¥ CNY            (所有语言环境)
$ HKD            (所有语言环境)
₹ INR            (所有语言环境)
RM MYR           (所有语言环境)
```

## 📱 修改效果

### ActionSheet显示内容
修改后，货币选择器弹窗中将显示：
- **$ USD** (而不是 "美元" 或 "US Dollar")
- **¥ JPY** (而不是 "日元" 或 "Japanese Yen")
- **¥ CNY** (而不是 "人民币" 或 "Chinese Yuan")
- **$ HKD** (而不是 "港币" 或 "Hong Kong Dollar")
- **₹ INR** (而不是 "卢比" 或 "Indian Rupee")
- **RM MYR** (而不是 "马币" 或 "Malaysian Ringgit")

### 选中货币显示
主界面选中的货币也会显示为：
- **$ USD** (而不是复杂的名称)
- **¥ JPY** (而不是复杂的名称)
